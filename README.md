## ğŸ“ Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #8b5cf6;
    --secondary-color: #2d2d2d;
    --accent-color: #ff9500;
}
```

### Adding Products

Add new product cards in the HTML:

```html
<div class="product-card">
    <img src="assets/img-new.jpg" alt="Product Name">
    <h3>Product Name</h3>
</div>
```
