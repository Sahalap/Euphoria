# Euphoria E-Commerce Website

A modern, responsive e-commerce website built with HTML and CSS, designed to provide customers with an engaging and seamless shopping experience.

## 🎨 Project Overview

Euphoria is more than just an online store - it's a vibrant and engaging space for customers to explore and discover products that bring joy and value. The website features a clean, modern design that makes shopping simple and enjoyable.

## ✨ Features

- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and intuitive interface for easy navigation
- **Product Categories**: Well-organized sections for Men's and Women's fashion
- **Brand Partnerships**: Showcase of top fashion brands
- **Customer Reviews**: Feedback section to build trust
- **Promotional Banners**: Eye-catching promotional content
- **Interactive Elements**: Hover effects and smooth transitions

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

### File Structure

```
euphoria/
│
├── index.html          # Main HTML file
├── styles.css          # All CSS styling
├── assets/             # Images, icons, and logos
│   ├── bg-*.jpg        # Background images
│   ├── img-*.jpg       # Product images
│   ├── *.svg           # Icons and logos
│   └── *.jpg           # Brand logos
└── README.md           # This file
```

## 📱 Responsive Breakpoints

The website is optimized for three main breakpoints:

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## 📝 Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #8b5cf6;
    --secondary-color: #2d2d2d;
    --accent-color: #ff9500;
}
```

### Adding Products

Add new product cards in the HTML:

```html
<div class="product-card">
    <img src="assets/img-new.jpg" alt="Product Name">
    <h3>Product Name</h3>
</div>
```
